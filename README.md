# 宾馆管理系统 python877

## 项目概述

宾馆管理系统是基于 Django + Python 开发的，采用 B/S 架构，主要用于宾馆的客房预约管理。

## 技术栈

- Python
- Django 框架
- B/S 结构
- MySQL 数据库

## 系统角色

### 管理员

- **客房类型管理**
  - 定义
  - 编辑
  - 删除
- **热门客房管理**
  - 设置
  - 更新信息
- **客房预约管理**
  - 审核
  - 查看
  - 删除预约记录
- **会员管理**
  - 账号信息增删改查
  - 处理用户咨询
- **评论管理**
  - 审核评论
  - 删除评论
- **系统配置**
  - 参数设置
  - 公告发布

### 会员

- **个人信息管理**
  - 查看
  - 修改资料
- **客房预订**
  - 浏览客房
  - 提交预约申请
  - 发起咨询请求
- **评论功能**
  - 发表评论
- **系统公告查看**
  - 查看公告
